# 单例模式

## 介绍

> - 系统中被唯一使用
> - 一个类只有一个实例

### 说明

> - 单例模式需要用到 java 的特性（private）
> - ES6 中没有(TypeScript 除外)
> - 只能用 JAVA 代码来演示 uml 图的内容

## 演示

### 实例

> - 登录框
> - 购物车

## 场景

> - JQuery 只有一个\$

```javascript
// jQuery 只有一个\$ -
if (window.jQuery !== null) {
  return window.jQuery;
} else {
  //　初始化
}
```

> - 模拟登录框
